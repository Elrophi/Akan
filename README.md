# Akan Names

```This is a Form that allows one to pick their birthday on a date picker and it displays their Akan name depending on the day which they were born.```

### Installation and setup
1. Clone to Local Machine on terminal or copy the link from github
1. Use any compiler for HTML,CSS AND JAVASCRIPT.
1. Use the Live Server to run the live site
1. Done
